# MSV Nosy‑Be — Template e‑commerce (statique)

Mini-site e‑commerce **100% statique** (HTML + Tailwind CDN + JS).

✅ Panier simulé (localStorage)  
❌ Pas de paiement réel (à connecter à Stripe/WooCommerce)

## Pages
- index.html (home)
- shop.html (listing + filtres)
- product.html?id=tk-noir-14-15 (fiche produit via query param)
- cart.html (panier)
- checkout.html (checkout démo)
- about.html / contact.html / b2b.html

## Déploiement
- Ouvrir `index.html` en local, ou publier sur Netlify / Vercel / GitHub Pages.
